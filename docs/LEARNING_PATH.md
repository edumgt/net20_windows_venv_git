# LEARNING_PATH (완전 입문자용)

## 0단계: 용어부터(01~03)
- 01: IP/Port는 무엇인가? (DNS로 IP 확인)
- 02: 로컬 IP/호스트 이름 확인
- 03: URL 구조(스킴/호스트/포트/경로/쿼리)

## 1단계: TCP 소켓 기초(04~07)
- 04: TCP 접속 테스트(포트 열렸나?)
- 05: 여러 포트 빠르게 체크(주의: allowlist만!)
- 06: 타임아웃이 왜 중요한지
- 07: 간단한 "배너" 읽기(서비스가 처음 보내는 문자열)

## 2단계: HTTP 기초(08~11)
- 08: urllib로 HTTP GET
- 09: requests로 HTTP GET(더 쉬움)
- 10: 헤더/상태코드/인코딩 보기
- 11: JSON API 호출/파싱

## 3단계: 로컬 서버로 이해(12~17)
- 12: 로컬 HTTP 서버(가장 쉬운 서버)
- 13: 로컬 HTTP 클라이언트
- 14: 파일 다운로드(스트리밍)
- 15: TCP 에코 서버(소켓 서버)
- 16: TCP 에코 클라이언트
- 17: UDP 한 번 보내보기(연결 없는 통신)

## 4단계: 운영 관점(18~20)
- 18: 재시도(backoff) 기본
- 19: 간단 모니터링(주기적으로 체크하고 로그 남기기)
- 20: asyncio로 여러 대상 동시에 접속(맛보기)
